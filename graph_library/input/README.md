# Input Data

This folder contains source data files used by the graph library build pipeline.

## RSST Unavoidable Configurations

Place the `unavoidable.conf` file here before building the graph library.

This file contains the 633 configurations from Robertson, Sanders, Seymour &
Thomas (1997). It is available from the original authors' proof materials at:

  https://people.math.gatech.edu/~thomas/FC/fourcolor.html

Used by: `modules/generate_rsst_unavoidable_graphs.py`

## McCooey Polyhedra Data

Three CSV files scraped from David McCooey's Visual Polyhedra database
(https://dmccooey.com/polyhedra/) containing exact 3D coordinates for 139
convex polyhedra across 5 categories:

| File | Rows | Description |
|---|---|---|
| `mccooey_polyhedra.csv` | 139 | One row per polyhedron — name, category, V/E/F counts, vertex configuration, edge list |
| `mccooey_vertices.csv` | 3,690 | One row per vertex — polyhedron foreign key, vertex ID, x/y/z coordinates (15 sig figs) |
| `mccooey_faces.csv` | 3,728 | One row per face — polyhedron foreign key, face ID, n-sides, semicolon-separated vertex cycle |

Categories: 5 Platonic, 16 Archimedean, 15 Catalan, 91 Johnson, 12 Prism/Antiprism.
All 139 polyhedra pass Euler's formula (V − E + F = 2).

Generated by: `tools/scrape_mccooey.py`  
Used by: `modules/generate_3D_graphs.py`  
Full documentation: `tools/README_scrape_mccooey.md`
